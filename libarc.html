LIBRARIES

Libraries hold vast amounts of information and data. I am interested in helping people find, use and evaluate this information through reference and instruction but also contextualizng, identifying, parsing, wrangling and analzing this data using programming technologies.



I'm experimenting with network visualization tools to visualize the subtlety and expertise that make reference work so valuable, since network visualization presents a way of conceptualizating relationships and the transmission of library resources.

The technologies applied are DHumanities, Node.XL, and d3.js. I sought out this project to learn network visualization basics, to understand the fundamentals of information visualization, to produce a tool to help faculty and students see relationships between library resources and build something aesthetically interesting.

I've sought out D3 Data-Driven documents, Introduction to Information Visualization By Riccardo Mazza and Visual Complexity: Mapping Patterns of Information by Manuel Lima

The test files are being tweeked but you can view slides of the work in progess here.

ARCHIVES

Archives embrace materials that lack a standardized structure. These materials come in all sizes, formats, and physical composition. I am interested in the digital tools which enable discovery, aggregration and dissemintation of these materials and their context.



I have used these content management systems in both library and archival practice for curating, organizing, modifying and creating digital content for web accessibility.
